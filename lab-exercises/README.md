# Lab Exercises

This directory contains coding exercises from AlgoProg lab sessions.

## Organization

Exercises can be organized by:
- **Lab Session**: `lab-01/`, `lab-02/`, etc.
- **Topic**: `sorting/`, `searching/`, `data-structures/`, etc.
- **Date**: `2024-01-15/`, `2024-01-22/`, etc.

## Example Structure

```
lab-exercises/
├── lab-01/
│   ├── problem1.cpp
│   ├── problem2.cpp
│   └── README.md
├── lab-02/
│   ├── problem1.py
│   └── README.md
└── README.md (this file)
```

## Adding Solutions

For each problem:
1. Create a file with descriptive name (e.g., `bubble-sort.cpp`, `binary-search.py`)
2. Add comments explaining the algorithm/approach
3. Include time and space complexity if applicable
4. Optionally add test cases or example inputs/outputs
